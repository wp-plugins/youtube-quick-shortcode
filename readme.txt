== Description ==

<strong>YouTube quick shortcode plugin contains</strong> -
<ul>
<li>Embedding YouTube videos using YouTube's URL address.</li>
<li>Text button "YouTube" within TinyMCE editor that outputs [ytube_shortcode url="|||||" ]</li>
<li>The admin user can embed clips from YouTube simply by pasting the URL address of the clip between any two | characters, i.e. [ytube_shortcode url="||https://www.youtube.com/watch?v=Hw_O_pH_mhI|||" ]</li>
<li>In order to work, the plugin settings page must be opened and prefered options saved. Note that even if the default settings are fine, the admin user still have to save the options.</li>
<li>The plugin is light and its main purpose is to allow the admin to embed a YouTube video with least clicks possible.</li>
<li>The plugin doesn't allow customization per post/page. It uses general settings and it is intended for users that prefer general, rather than strongly customizable, settings.</li>
<li>The plugin's' general settings lets the user define screen size, show information, show related, show controls of the YouTube clips.</li>
</ul>
The main purpose of the plugin is to save clicks when a Wordpress administrator wants to embed a YouTube clip in a Wordpress site.

== Installation ==

1. Upload ytube-quick-shortcode.zip` onto your local computer.
2. Go to your WordPress Dashboard and select <strong>Plugins >> Add New</strong>.
3. Click on the <strong>Upload</strong> option at the top and select the ytube-quick-shortcode.zip file` you just downloaded.
4. Click on <strong>Install</strong>.
5. Activate the plugin through the 'Plugins' menu in WordPress.
6. In order to work, the plugin settings page must be opened and prefered options saved. Note that even if the default settings are fine for you, you still have to save the options.
7. Within your TinyMCE editor click on YouTube text button and paste URL of YouTube clip between "  "
